Kmeans(K, R){
  Choisir K centroides (c1, ..., ck)
  creer k clusters vides (cl1, ..., clk)
  tant que pas condition d'arret*{
    pour chaque ligne de R{
      calculer distances au centroides
      mettre dans le bon cluster en fonction de la distance
    }
    mettre a jour les centroides de chaque clusters
  }
}


*1- nb d'iteration max
 2- aucun changement dans les clusters
 3- difference de distance entre les centroides faible
